## 📖 Sobre o Projeto
Este projeto consiste numa aplicação web desenvolvida em **Django**, concebida para funcionar como um portfólio académico e profissional. A aplicação permite a gestão e visualização de:
* Licenciaturas e Unidades Curriculares
* Projetos desenvolvidos
* Tecnologias dominadas
* Competências (Hard e Soft Skills)
* Trabalhos Finais de Curso (TFCs)

## ⚙️ Tecnologias Utilizadas
* **Linguagem:** Python
* **Framework:** Django
* **Base de Dados:** SQLite (default)
* **Controlo de Versões:** Git & GitHub
